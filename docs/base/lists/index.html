Simple list modifier to remove margin and bullets from a list.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.

Installation


  npm install --save-dev scales-base-lists

or


  bower install --save-dev scales-base-lists

Usage

Sometimes you need a list without any margin or padding and no bullets.

Bare List


  <ul class="list--bare">
    <li>Scales</li>
    <li>Patterns</li>
    <li>Lists</li>
  </ul>

Other uses

Since there are times when you might want to make a list bare without using a class, list--bare is also able to be used as a mixin or extended as a placeholder.


  @include list--bare;

  @extend %list--bare;

Available Classes

  • .list--bare
