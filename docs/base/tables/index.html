These are the base styles for creating tables.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.

Installation


  npm install --save-dev scales-base-tables

or


  bower install --save-dev scales-base-tables

Usage

Tables can get pretty crazy. Scales provides classes related to cell spacing, borders, stripes, and numerical values.

Default table

The default doesn't require any classes. It makes tables 100% width and applies some default vertical and text alignment for colspan and rowspan.

You can also use the .numerical class to right align numerical values in the cell.


  <table>
    <thead>
      <tr>
        <th>Lorem</th>
        <th>Ipsum</th>
        <th>Dolor</th>
        <th>Sit</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <th rowspan="3">Sit</th>
        <td>Dolor</td>
        <td class="numerical">03.788</td>
        <td>Lorem</td>
      </tr>
      <tr>
        <td>Dolor</td>
        <td class="numerical">32.210</td>
        <td>Lorem</td>
      </tr>
      <tr>
        <td>Dolor</td>
        <td class="numerical">47.797</td>
        <td>Lorem</td>
      </tr>
      <tr>
        <th rowspan="2">Sit</th>
        <td>Dolor</td>
        <td class="numerical">09.640</td>
        <td>Lorem</td>
      </tr>
      <tr>
        <td>Dolor</td>
        <td class="numerical">12.117</td>
        <td>Lorem</td>
      </tr>
    </tbody>
  </table>

Fixed cell width styles

Makes the table cells fixed width using table-layout: fixed. How it works. Can be used in conjunction with Size Overrides.


  <table class="table--fixed table--bordered">
  <thead>
    <tr>
      <th class="size-10">Lorem</th>
      <th class="size-20">Ipsum</th>
      <th class="size-20">Dolor</th>
      <th class="size-50">Sit</th>
    </tr>
  </thead>
  <tbody>
  ...
  </tbody>
  </table>
