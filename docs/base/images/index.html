These are some simple styles for fixing images so they are a bit more responsive. This adds some styles to images to make them scale with the viewport size unless a width or height are specified and fixes a bug with google maps.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.

Installation


  npm install --save-dev scales-base-images

or


  bower install --save-dev scales-base-images
