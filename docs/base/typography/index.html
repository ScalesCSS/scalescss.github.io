Base styles for typographic elements in Scales.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.

Installation


  npm install --save-dev scales-base-typography

or


  bower install --save-dev scales-base-typography

Usage

Scales provides default typography for headings, text elements, lede paragraphs, and code blocks.

Lede Paragraph

The lede paragraph is styled slightly larger than the base font size (1.25rem instead of 1rem).


  <p class="lede">Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed gravida urna quis nulla ultrices, sed efficitur risus elementum. Vivamus semper ex a lorem sodales, rhoncus rhoncus augue cursus. Sed vestibulum felis vel euismod aliquet. Suspendisse sit amet euismod nibh, eget fermentum arcu. Aliquam lacinia, sem eu ultricies auctor, velit nunc commodo sapien, pharetra maximus augue dui eget dui.</p>

Code Block


  <pre><code>
    code goes here
  </code></pre>

Available Classes

  • .lede or .lead

Heading Utility Classes

  • .h1 , .alpha
  • .h2 , .beta
  • .h3 , .gamma
  • .h4 , .delta
  • .h5 , .epsilon
  • .h6 , .zeta

Available Variables

  • $base-text-color : #333333
  • $code-block-padding : 0 1em 1em
  • $code-block-bg-color : #444444
  • $code-block-text-color : #ffffff
  • $base-font-stack : "Helvetica Neue", helvetica, arial, sans-serif
  • $heading-font-stack : Times, "Times New Roman", serif
  • $heading-font-weight : normal
  • $base-font-size : 1rem
  • $base-line-height : 1.5
  • $lede-font-size : 1.25rem
  • $h1-size : 2.25rem
  • $h2-size : 2rem
  • $h3-size : 1.75rem
  • $h4-size : 1.5rem
  • $h5-size : 1.25rem
  • $h6-size : 1rem
