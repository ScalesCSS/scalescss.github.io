These are the base styles for block quotes.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.

Installation


  npm install --save-dev scales-base-lists

or


  bower install --save-dev scales-base-lists

Usage

A nice semantic way to markup a blockquote is using the `figure` element. See A List Apart for the rationale behind this approach.

Default blockquote


  <figure>
    <blockquote class="blockquote">
      <p class="blockquote__quote">For me, it is far better to grasp the Universe as it really is than to persist in delusion, however satisfying and reassuring.</p>
    </blockquote>
    <figcaption class="blockquote__source">Carl Sagan, <cite>The Demon-Haunted World</cite></figcaption>
  </figure>
