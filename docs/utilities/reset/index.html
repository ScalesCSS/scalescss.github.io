The Scales Reset Utility is a companion to Normalize.css that resets a few things that are still useful in a traditional reset. Borrows from HTML5 Boilerplate and Reset Restarted.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.

Installation


  npm install --save-dev scales-utilities-reset

or


  bower install --save-dev scales-utilities-reset

Usage

Should be loaded immediately after _normalize.scss.

Available Variables

  • $selection-bg-color
  • $selection-text-color
