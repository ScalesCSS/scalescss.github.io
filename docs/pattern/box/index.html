A simple pattern for boxing off content and separating it visually from the surrounding content.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.
  • Scales Clearfix

Installation


  npm install --save-dev scales-patterns-box

or


  bower install --save-dev scales-patterns-box

Usage

The box pattern can be used on any element that you want. Scales gives several classes to control the amount of padding in the box.

Default Box

The default padding on the Default Box is 1em.


  <div class="box">
    The domestic cat is a small, usually furry, domesticated, and carnivorous mammal. It is often called the housecat when kept as an indoor pet, or simply the cat when there is no need to distinguish it from other felids and felines.
  </div>

Other uses

Since there are times when you want any element to be boxed off, possibly with different amounts of padding at different breakpoints, the box pattern is also able to be used as a mixin. Simply pass in the padding value you want to use.


  @include box($padding);

Available Classes

  • .box
  • .box--flush
  • .box--tiny
  • .box--small
  • .box--large
  • .box--huge

Available Variables

  • $box-padding
  • $box-padding-tiny
  • $box-padding-small
  • $box-padding-large
  • $box-padding-huge
