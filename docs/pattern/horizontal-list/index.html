A pattern for making a list horizontal instead of vertical. This pattern is particularly useful for various types of navigation, like your site's main navigation, breadcrumbs, and tag lists.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.

Installation


  npm install --save-dev scales-patterns-horizontal-list

or


  bower install --save-dev scales-patterns-horizontal-list

Usage

Horizontal List should be used whenever you need to make an ordered or unordered list horizontal instead of vertical.

You can extend it with --delimited to semantically mark up a character delimited list for things like tags, etc.

Scales also provides options to center, right align, or have the list fill the available space.

Default Horizontal List

The default simply makes the list horizontal and left aligned.


  <ul class="horizontal-list">
    <li>Sass</li>
    <li>CSS</li>
    <li>HTML</li>
  </ul>

Character Delimited Horizontal List

The default delimiter is a comma. This can be overridden by the $horizontal-list-delimiter variable.


  <ul class="horizontal-list horizontal-list--delimited">
    <li>Sass</li>
    <li>CSS</li>
    <li>HTML</li>
  </ul>

Centered Horizontal List

Centers the list in its container.


  <ul class="horizontal-list horizontal-list--center">
    <li>Sass</li>
    <li>CSS</li>
    <li>HTML</li>
  </ul>

Right Aligned Horizontal List

Right aligns the list in its container


  <ul class="horizontal-list horizontal-list--right">
    <li>Sass</li>

Fill Style Horizontal List

This forces the list items to take up all of the available space. The content in the list item is centered. You can use horizontal-list-right on the list items to change this default.


  <ul class="horizontal-list horizontal-list--fill">
    <li>Sass</li>
    <li>CSS</li>
    <li>HTML</li>
  </ul>

Other uses

Since there are times when you want a list to be horizontal at some media queries and vertical at others, the horizontal list pattern is also able to be used as a mixin or extended as a placeholder.


  @include horizontal-list;

  @extend %horizontal-list;

Available Classes

  • .horizontal-list--delimited
  • .horizontal-list--center
