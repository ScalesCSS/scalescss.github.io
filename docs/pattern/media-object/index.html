DISCLAIMER: For more control, you'll probably want to use the Flag Pattern. The Media Object is mostly kept around for historical purposes.

A simple pattern for placing variable amounts of content next to a fixed image or other media. This is based on the Media Object created by Nicole Sullivan.

The media object is probably the most useful OOCSS abstraction. Read more here: http://stubbornella.org/content/2010/06/25/the-media-object-saves-hundreds-of-lines-of-code.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.
  • Scales Clearfix

Installation


  npm install --save-dev scales-patterns-media-object

or


  bower install --save-dev scales-patterns-media-object

Usage

Default Media Object

The default puts an image, or other media, to the left of some content.


  <div class="media">
      <a class="media__figure" href="http://scalescss.github.io/scales">
          <img src="http://placekitten.com/g/150/150/">
      </a>
      <div class="media__body">
          <p>The domestic cat is a small, usually furry, domesticated, and carnivorous mammal. It is often called the housecat when kept as an indoor pet, or simply the cat when there is no need to distinguish it from other felids and felines.</p>
      </div>
  </div>

Using the <figure> element.

Maybe you want to use the <figure> element instead? Sure thing. The classes are the same.


  <figure class="media">
      <a class="media__figure" href="http://scalescss.github.io/scales">
          <img src="http://placekitten.com/g/150/150/">
      </a>
      <figcaption class="media__body">
          <p>The domestic cat is a small, usually furry, domesticated, and carnivorous mammal. It is often called the housecat when kept as an indoor pet, or simply the cat when there is no need to distinguish it from other felids and felines.</p>
      </figcaption>
  </figure>

Available Classes

  • .media
  • .media--rev
  • .media__figure
  • .media__body
  • .media--stacked

For changing the space between the image and the content

  • .media--flush
  • .media--tiny
  • .media--small
  • .media--large
  • .media--huge

  • $media-gutter-tiny
  • $media-gutter-small

  • $media-gutter-huge
