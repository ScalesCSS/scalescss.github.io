A pattern for making a list vertical after it's been made horizontal.

Dependencies

  • Sass - Either Libsass or Ruby Sass.
  • NPM or Bower - To install and manage the Scales repositories. Not technically required as you could manually install.

Installation


  npm install --save-dev scales-patterns-vertical-list

or


  bower install --save-dev scales-patterns-vertical-list

Usage

Vertical List should be used whenever you need to make an ordered or unordered list vertical instead of horizontal.

You can extend it with --delimited to semantically mark up a character delimited list for things like tags, etc.

Default Vertical List

The default simply makes a list vertical. It applies display: block to the <li>'s. Unless you have another display property on a list, this class isn't super useful.


  <ul class="vertical-list">
    <li>Sass</li>
    <li>CSS</li>
    <li>HTML</li>
  </ul>

Character Delimited Vertical List

The default delimiter is a comma. This can be overridden by the $vertical-list-delimiter variable.


  <ul class="vertical-list horizontal-list--delimited">
    <li>Sass</li>
    <li>CSS</li>
    <li>HTML</li>
  </ul>

Other uses

Since there are times when you want a list to be vertical at some media queries and horizontal at others, the vertical list pattern is also able to be used as a mixin or extended as a placeholder.


  @include vertical-list;

  @extend %vertical-list;

Available Classes

  • .vertical-list
  • .vertical-list--delimited

Available Variables

  • $vertical-list-delimiter
